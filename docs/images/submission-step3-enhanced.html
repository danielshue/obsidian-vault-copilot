# Study Buddy Agent ## Overview The Study Buddy Agent is a comprehensive learning companion that helps you organize study materials, track progress, and achieve your educational goals within Obsidian. ## Features - **Study Session Tracking**: Log and analyze your study sessions - **Goal Management**: Set and track learning objectives - **Progress Reviews**: Automated weekly and monthly progress reports - **Smart Linking**: Automatically links related study materials - **Spaced Repetition**: Suggests review schedules based on learning science ## Usage ### Starting a Study Session 1. Open the command palette (Ctrl/Cmd + P) 2. Search for "Start Study Session" 3. Choose your subject or topic 4. The agent will create a session note and timer ### Setting Learning Goals Use the "Set Learning Goal" command to define: - Subject or skill you want to learn - Target completion date - Milestones and checkpoints - Resources you'll use ### Reviewing Progress Run "Study Progress Review" to get: - Summary of completed study hours - Goal completion status - Suggested next steps - Areas needing more attention ## Example ``` > Start Study Session Subject: Advanced TypeScript Duration: 2 hours Topics: Generics, Conditional Types ``` The agent will create a structured note and track your session automatically.
