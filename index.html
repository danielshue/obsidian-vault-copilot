How It Works

1

Connect GitHub Copilot

Vault Copilot uses the GitHub Copilot CLI SDK for secure, structured AI interactions. Requires an active GitHub Copilot subscription.

2

Chat & Attach Context

Open the chat interface, attach relevant notes, and start a conversation. The AI has full awareness of your vault structure and content.

3

Extend with Skills

Install extensions from the catalog or create your own. Skills give the AI new capabilities like task management, journaling workflows, or external integrations.

Built on Modern Technology
